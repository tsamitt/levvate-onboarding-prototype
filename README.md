# Levvate Onboarding Prototype

This is a lightweight front-end prototype for a multi-step client onboarding system with saved progress. It was built as a fast proof of concept for an internship application focused on execution, workflow design, and CRM automation.

## What it demonstrates
- Multi-step onboarding flow
- Saved progress using browser localStorage
- Review step before submission
- Completion tracking and simulated backend events
- Example HubSpot payload mapping for client handoff

## How to run
1. Download the folder
2. Open `index.html` in a browser
3. Enter sample data and move through the steps
4. Refresh the page to confirm saved progress is preserved

## Production version would add
- Real authentication
- Database persistence instead of localStorage
- Server-side validation
- HubSpot API integration
- Stripe and scheduling integrations
- Team notifications through email or Slack

## Why this is relevant
The internship asks for a builder who can ship usable systems. This prototype focuses on one of the core requested workflows: a progress-tracking onboarding system that can later connect to HubSpot and backend actions.
